# Data Acquisition & Preprocessing

## 1. Data Source
The dataset used in this project is the **Heart Disease UCI Dataset** from the UCI Machine Learning Repository.
- **Dataset ID:** 45
- **Features:** 13 clinical features (e.g., age, sex, blood pressure, cholesterol).
- **Target:** Presence of heart disease (originally multi-class 0-4, binarized to 0 and 1).

## 2. Exploratory Data Analysis (EDA)
EDA was performed in the `notebooks/eda_and_modeling.ipynb` file. Key steps included:
- **Target Distribution:** Analyzing the balance of the binary classes.
- **Correlation Heatmap:** Identifying highly correlated features to understand relationships.
- **Feature Distribution:** Analyzing the spread of numerical features like age and cholesterol.

## 3. Data Preprocessing Pipeline
To ensure reproducibility, a `ColumnTransformer` pipeline was built in `src/data_preprocessing.py`:
- **Numerical Features:** Handled missing values with `SimpleImputer(strategy='median')` and scaled using `StandardScaler`.
- **Categorical Features:** Handled missing values with `SimpleImputer(strategy='most_frequent')` and encoded using `OneHotEncoder`.

The fitted preprocessor is saved as `models/preprocessor.joblib` and is reused during inference to guarantee identical data transformation.
