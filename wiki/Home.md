# Heart Disease Prediction MLOps Project

Welcome to the detailed Wiki for the Heart Disease Prediction MLOps project. This project is part of Assignment 01 and demonstrates a complete, production-ready Machine Learning Operations lifecycle.

## Project Overview

The goal of this project is to build a machine learning classifier to predict the risk of heart disease based on patient health data, and deploy the solution as a cloud-ready, monitored API.
