FAQs
Q. Which dataset should be used?
The assignment requires you to use the Heart Disease UCI Dataset from the UCI
Machine Learning Repository.
Q. What kind of Exploratory Data Analysis (EDA) is expected?
You should include:
•
•
•
•
•

Histograms
Correlation heatmaps
Class distribution plots
Missing value analysis
Feature relationship analysis

Professional visualizations using Matplotlib, Seaborn, or Plotly are recommended.
Q. Should we clean and preprocess the data?
Yes. You should:
•
•
•
•

Handle missing values
Encode categorical variables
Scale numerical features (if applicable)
Split train/test data properly

Q. How many models should be trained?
At least two classification models are expected.
Examples:
•
•
•
•

Logistic Regression
Random Forest
XGBoost
Support Vector Machine or any choice of yours
